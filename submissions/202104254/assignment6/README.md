머신러닝 기반 이민 국가 추천 및 정책 리스크탐지 시스템


실행방법 - 코드 실행 시 폴더에 final.zip에 동봉된 lgbt-military-equaldex.csv, 한국 현황 정리.csv 파일 삽입 필요

## 1. 프로젝트 개요
* **연구 배경:** 서구 선진국은 경제 성장과 인권 포용성이 비례하나, 한국은 경제 지표(GDP) 대비 인권 의식이 낮은 '문화적 지체' 현상이 관찰됨.
* **연구 목적:**
    1.  경제·거버넌스 지표로 LGBT 군 복무 합법성 예측 모델 구축.
    2.  정량적 예측(지표)과 정성적 여론(뉴스 감성)을 비교해 '이민 리스크 국가' 탐지 및 경고 시스템 구현.

## 2. 시스템 아키텍처 및 방법론
* **데이터 수집:**
    * **정량:** World Bank Open Data (GDP, 군비, 정부 효과성) 및 Equaldex (법적 상태) 활용.
    * **정성:** 주요 국가의 LGBT 관련 영문 뉴스 헤드라인 및 요약문.
* **모델 설계:**
    * **Model A (정책 예측):** `SGDClassifier` (Loss='log_loss') 활용, 점진적 학습 및 과적합 방지 적용.
    * **Model B (여론 분석):** `DistilBERT` 활용, 뉴스 텍스트의 긍/부정 감성 분류 및 점수화.
* **시스템 구조:**
    * **입력:** 사용자 지표 입력 또는 프리셋 국가 선택.
    * **추론:** 이중 엔진(Dual Inference) 가동 (SGD + BERT).
    * **판정:** 예측 결과와 감성 점수를 종합해 리스크 등급(Safe/Hidden Risk/Critical Risk) 산출.

## 3. 연구 결과
* **정량적 성능:** `SGDClassifier` 기준 ROC-AUC **0.8448**, 정확도 **76.92%** 달성.
* **사례 분석:**
    * **🇩🇪 독일:** 지표 우수, 여론 긍정 → **Safe Match (일치)**.
    * **🇨🇳 중국:** 지표 양호(Legal 예측), 여론 부정 → **Mismatch (불일치)**.
    * **🇰🇷 한국:** 경제 지표 선진국(Legal 예측), 여론 부정적(Negative) → **Mismatch (불일치)**.
* **CLI 시뮬레이션:** 5회 테스트 결과, 정부 효과성이 높을수록 허용 확률이 증가하고 과도한 군비 지출은 비허용 요인으로 작용함을 확인.

## 4. 결론 및 제언
* **의의:** 모델의 오분류(Mismatch)를 단순 오류가 아닌 **'사회적 리스크 신호'**로 재해석함. [cite_start]GDP 수치 이면의 사회적 차별 가능성을 데이터로 입증.
* **한계:** 결측치 평균 대체에 따른 특성 희석, 이진 분류(Legal/Not Legal)의 단순함, 영문 뉴스 데이터 의존.
* **향후 계획:** 멀티클래스(LGB permitted 등) 확장, 멀티모달(정형+비정형) 통합, 결측치 처리 개선, 인과추론 접목, 웹 대시보드(Streamlit 등) 고도화 추진.
